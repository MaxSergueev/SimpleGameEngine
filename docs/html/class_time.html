Detailed Description

Static utility class for frame timing and delta time management.

Time provides frame rate control and delta time calculations for consistent game loop timing. It manages frame delays to maintain target FPS and calculates delta time between frames for frame-rate independent animations and physics. Uses SDL timing functions for cross-platform compatibility.
