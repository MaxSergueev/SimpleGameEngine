Detailed Description

Physics-based bowling ball actor for bowling game mechanics.

BowlingBall represents a heavy spherical object that can be rolled in a specified direction with realistic physics properties. It combines visual representation, movement mechanics, and collision detection to simulate a bowling ball's behavior. The ball can be controlled programmatically and includes mass properties for potential physics interactions with other objects.
