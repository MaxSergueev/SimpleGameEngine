Detailed Description

SDL window wrapper for cross-platform window management.

Window encapsulates SDL window creation and management, providing a simple interface for creating OpenGL-compatible windows. It handles SDL initialization, window creation with specified dimensions, and proper cleanup of SDL resources. The window is automatically configured for OpenGL rendering contexts.
