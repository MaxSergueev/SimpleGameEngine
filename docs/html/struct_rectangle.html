Detailed Description

2D rectangle structure for UI and sprite rendering

Rectangle represents a 2D rectangular area defined by position and dimensions. It provides conversion utilities for SDL rendering and comparison operators for rectangle equality testing. Used primarily for sprite source rectangles and UI element positioning.
