Detailed Description

Component that provides physics simulation for actors.

RigidBodyComponent implements basic physics behavior including velocity, acceleration, mass, and gravity. It handles collision response by calculating penetration vectors and reflecting velocity along collision normals. Automatically creates an AABB collider if none exists on the actor.
