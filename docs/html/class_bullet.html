Detailed Description

Projectile actor for shooting mechanics in the game engine.

Bullet represents a projectile that can be fired by actors (players, enemies, etc.). It handles movement, collision detection, damage dealing, and automatic cleanup after a specified lifetime. Bullets are designed to be pooled for performance, allowing reuse without constant allocation/deallocation. They automatically detect hits against players, enemies, and solid objects.
