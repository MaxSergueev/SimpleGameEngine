Detailed Description

Component that renders 2D sprites for actors.

SpriteComponent provides 2D sprite rendering functionality by managing textures and draw order. It automatically registers with the scene's renderer and handles texture flipping for directional sprites. The component calculates proper origin points for centered sprite rendering and integrates with both SDL and OpenGL renderers.
