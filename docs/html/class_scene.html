Detailed Description

Base class for game scenes that manage actors, rendering, and collision detection.

Scene provides the foundation for game states, managing collections of actors, collision detection between AABB colliders, and integration with the rendering system. It handles actor lifecycle management with pending/dead actor queues to safely manage objects during update cycles.
