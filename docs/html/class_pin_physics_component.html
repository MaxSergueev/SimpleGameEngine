Detailed Description

Specialized physics component for bowling pin behavior.

PinPhysicsComponent provides friction and momentum simulation for bowling pins. It applies dynamic friction based on movement speed to create pin physics where pins gradually slow down after being knocked. The component features speed-dependent friction coefficients and automatic stopping for very slow movements to prevent infinite sliding.
